## Iperf 測試Command
### Tx 測試

#### TCP   
```
PC : iperf  -s  -i 1
BOX : iperf  -c PC_IP -t 60 -i 1 -P 2 -l  64000
```

#### UDP
```
PC: iperf -s -u -i 1
BOX: iperf -c PC_IP -t 60 -i 1 -l 1024 -b 50M -u
```

### Rx 测试  

#### TCP  
```
BOX : iperf  -s  -i 1
PC : iperf  -c BOX_IP -t 60 -i 1 -P 2 -l  64000
```
